Hello RND Framework!

RND Framework is a modern web developing tool that includes an automatic page renderer which is written by PHP. It helps you to convert static web pages to a dynamic form easily, faster, and safer. In short, you can develop your websites rapidly with RND Framework!

H tml5up, Bootstrap, Materialize, Pure.css or any other front-end frameworks will run here!

Based on W3Schools lessons!

W3Schools is the best online portal to learn HTML, CSS, and JavaScripts. Default RND Framework follows W3.CSS styles & standards that you can easily change as your project needs.

Any w3-class-name can be used in RND Framework by replacing prefixes as rnd-class-name.
For example, w3-container class is used to create a container in w3schools. In RND Framework, we use rnd-container class to achieve it.

Created to save your time!

Web developers have to work on different websites from time to time. Remembering those codes & styles are really difficult and also take a lot of hours. RND Framework is developed to minimize developing time while managing codes & styles to their best performance. It includes followings,

Templates

Templates are all about the look & feel of your website. RND Framework provides you facility to create/modify your templates as well as importing/exporting them between your projects easily.

Layouts

Layouts are just like child-templates which are independent from the main template. You can create/edit them to render special pages in your project. And also you can transfer them to other projects easily. For example, if you create a layout that has a contact form, you can easily copy/paste it to other RND websites and render the contact form ASAP. No need to worry about editing styles, or matching containers.

Plugins

Registering a new user, logging in to the dashboard are technical functions which need programming skills. RND Framework simply uses plugins to achieve these difficult tasks and it helps beginners to avoid bugs, security and performance issues on their websites. Plugins are also moveable, so you can use them in your other RND websites easily.
