Important Tips:

A developer who uses RND Framework should know the following as key features.

Templates are universal designs!

The styles, scripts you add to your main template will universally affect all pages in your website. You can use Layouts to localize special styles, scripts in special pages. This will dramatically boost your website’s speed and SEO!

Layouts will overlap on styles/scripts in the main template!

Page layouts are just like child templates. Their styles/scripts will be in your website, even if you change your template in future. As styles/scripts in layouts load after main styles/scripts, they can overlap on main template’s styles, scripts, and variables.

Plugins are more powerful than anything!

Connecting a cloud database, creating user profiles, rendering a dashboard as well as collecting a payment can be done via special plugins. Plugins can include their own independent Page Layouts and Templates to smoothen UI and UX. Whatever task you need to do via your website, it always needs a plugin.

Organize media files in upload folder

Your uploaded images, videos and other files should be stored in the upload folder in rnd-content folder. It will help you to run backups, import/export processes in future easily & faster.
